# 기본 사항

- 이 프로젝트는 반응형 웹 서비스입니다.

# 기술 스택

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Backend/Auth:** Supabase (@supabase/ssr 사용)
- **Styling:** Tailwind CSS
- **Database:** PostgreSQL (Supabase)

# 코드 작성 규칙

- 모든 코드는 Best Practices에 따라 작성되어야 합니다.
- 모든 코드는 최대한 간결하고 가독성 좋게 작성되어야 합니다.
- 모든 코드에는 적절한 주석이 포함되어야 합니다.
- 코드 중복을 피하고, 재사용 가능한 컴포넌트와 함수를 만들어야 합니다.

# 기본 기능

- 로그인 없이 사용가능한 웹 서비스입니다.
- Supabase를 활용한 구글, 네이버, 카카오 소셜 로그인을 지원합니다.
