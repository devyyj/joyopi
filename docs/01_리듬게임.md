# Rhythm Cat 게임 기획서

## 기본 정보
- 게임 이름: **Rhythm Cat**
- 목적: 무한 스테이지를 클리어하며 최고 점수를 노리는 리듬 게임
- 음악 없음 (박자감 집중)

## 캐릭터 / 이미지
- **노트 · 히트존**: `public/cat/pow1.png`, `pow2.png`, `pow3.png` (레인별 1~3)

## 게임 진행 구조
```
[준비 2초] → [스테이지 1 · 30초 · 50 BPM] → [전환 3초] → [스테이지 2 · 30초 · 55 BPM] → …
```
- **스테이지**: 1부터 무한 증가, 각 30초
- **BPM**: 스테이지 1 = 50 BPM, 이후 스테이지마다 +5
- **전환**: 별도 화면 없음 — 노트 생성 3초 중단 후 재개, 인게임 카운트다운 표시
- **조작**: PC(A/스페이스/D 키, 마우스 클릭), 모바일(터치)

## 레인 시스템
- **3레인**: 화면을 3등분, 각 레인 독립 히트존
- 키보드: A키 = 1레인, 스페이스/중앙클릭 = 2레인, D키 = 3레인
- 터치: 화면 X 위치로 레인 자동 감지

## 노트 시스템
- 노트: pow 이미지를 원형으로 표현 (레인별 pow1/2/3)
- 16분음표 그리드, 58% 확률 생성, 연속 최대 4개 제한
- 노트는 화면 상단에서 히트존까지 내려옴

## 점수 시스템
- 퍼펙냥! (Perfect ≤70ms): **+10점 + 콤보보너스**
- 좋아옹! (Good ≤150ms): **+5점 + 콤보보너스**
- 놓쳤냥 (Miss): **-5점**, 콤보 초기화
- 빈 레인 클릭/터치 = 미스 (연타 방지)
- 콤보: 1콤보당 +1점 추가 보너스

## 효과음
- 점수 획득 시: `meow1.mp3` 또는 `meow2.mp3` 랜덤 재생
- 미스 시: `purr.mp3` 재생

## UI/UX 설계 기준
- **전체 테마**: 밝은 테마, 슬레이트 강조색 (`#64748B`)
- **언어**: 모든 UI 텍스트 한글 사용 (BPM/STAGE 제외)
- **좌측 상단**: STAGE 번호(대형), 현재 BPM
- **중앙 상단**: 남은 시간 (대형), 전환 시 다음 스테이지 카운트다운
- **우측 상단**: 현재 점수 (대형)
- **진행 바**: 상단, 현재 스테이지 진행률
- **타이머 경고**: 10초 이하 빨간색 + 글로우
- **그만 버튼**: 우측 하단
