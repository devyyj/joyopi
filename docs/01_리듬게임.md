# Rhythm Cat 게임 기획서

## 기본 정보
- 게임 이름: **Rhythm Cat**
- 목적: 스테이지 1, 2를 클리어하며 최고 점수를 노리는 리듬 게임
- 음악 없음 (박자감 집중)

## 캐릭터
- **버키 (Bucky)**: `public/cat/bucky.png` - 회색 고양이, 슬레이트 컬러 테마
- 노트와 히트존 이미지로 버키 사용

## 게임 진행 구조
```
[준비 2초] → [스테이지 1 · 30초 · 50 BPM] → [휴식 5초] → [스테이지 2 · 30초 · 80 BPM] → [결과]
```
- **스테이지 1**: 50 BPM, 30초
- **휴식**: 5초 — 카운트다운 표시, 스테이지 2 BPM 안내
- **스테이지 2**: 80 BPM, 30초
- 점수는 전 스테이지 통산 누적
- 조작: PC(키보드 아무 키/마우스), 모바일(터치)

## 노트 시스템
- 노트: 버키 이미지를 원형으로 표현
- 16분음표 그리드, 60% 확률 생성, 연속 최대 4개 제한
- 노트는 화면 상단에서 히트존까지 내려옴

## 점수 시스템
- 퍼펙냥! (Perfect): **+10점**
- 좋아옹! (Good): **+5점**
- 놓쳤냥 (Miss): **-5점**
- 콤보: 1콤보당 +1점 추가 보너스, 팝업 애니메이션

## UI/UX 설계 기준
- **전체 테마**: 밝은 테마, 슬레이트 강조색 (`#64748B`)
- **언어**: 모든 UI 텍스트 한글 사용 (BPM 제외)
- **폰트**: 전체적으로 크고 명확하게
- **타이머**: 현재 스테이지 남은 시간 표시, 10초 이하 경고 색상
- **스테이지 표시**: 화면 상단에 현재 스테이지(1/2) 표시
- **진행 바**: 현재 스테이지 진행률 표시
- **휴식 화면**: 전체 캔버스에 카운트다운 + 스테이지 2 안내 (확실하게 표시)
- **결과 화면**: 최종 점수, 판정 통계(퍼펙냥/좋아옹/놓쳤냥), 최고 콤보, 다시하기 버튼
