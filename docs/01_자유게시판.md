# 자유게시판 (Free Board) - 상세 기능 명세

## 1. 개요
조요피 연구소의 메인 커뮤니티 공간으로, 즉각적인 반응성(Optimistic UI)과 풍부한 멀티미디어 지원을 특징으로 하는 피드형 게시판입니다.

## 2. 핵심 기능

### 2.1. 멀티 미디어 지원
- **사진 첨부**: 게시글당 **최대 10장**의 사진 첨부 가능.
- **이미지 최적화**: 업로드 전 클라이언트 사이드 리사이징(최대 너비 1200px, WebP 변환)을 통해 서버 부하 최소화.
- **그리드 레이아웃**: 첨부된 사진 개수에 따라 최적화된 그리드(1개: 4:3 비율, 2개 이상: 정사각형) 노출.

### 2.2. 무한 스크롤 (Infinite Scroll)
- 게시글 목록 조회 시 10개 단위로 페이징 조회. Native `Intersection Observer` 적용.

### 2.3. 플로팅 액션 버튼 (FAB)
- 화면 우측 하단에 고정된 시그니처 컬러(#e2ff00) 버튼 제공. '작성' 직관성을 높인 연필 아이콘과 호버 툴팁 적용.

### 2.4. 구조화된 레이아웃
- **Identity Bar**: 작성자 정보와 관리 기능을 상단에 배치하여 본문과 시각적으로 격리.
- **정보 위계 최적화**: [제목] -> [작성 시간] -> [본문] 순으로 시선이 자연스럽게 흐르도록 배치하여 가독성 향상.

### 2.5. 인라인 편집 (Inline CRUD)
- 게시글 수정 시 피드에서 즉시 입력 폼으로 전환. 사진 추가 및 삭제 기능 포함.
- `useOptimistic`을 통한 즉각적인 UI 반영 및 서버 액션 처리.

### 2.6. 본문 가독성 관리
- 제목: 최대 50자 제한, 목록에서 2줄 초과 시 생략 처리.
- 본문: 최대 5,000자 제한, 목록에서 5줄 초과 시 '더보기/접기' 기능 제공.

### 2.7. 답글(댓글) 시스템
- **한 줄 소통**: 간결한 소통을 위해 줄 바꿈이 금지된 한 줄 입력(Input) 방식 채택.
- **길이 제한**: 최대 200자 제한 및 실시간 카운터 제공.

### 2.8. 양방향 좋아요
- 게시글 및 개별 댓글에 대한 하트 토글 기능.
