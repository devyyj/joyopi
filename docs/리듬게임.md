# Rhythm Me 게임 기획서

## 기본 정보
- 게임 이름: **Rhythm Me**
- 목적: 60초 동안 점진적으로 빨라지는 비트를 정복하는 리듬 게임
- 음악 없음 (박자감 집중)

## 게임플레이
- **총 플레이 시간**: **60초** 고정
- **난이도 변화 (BPM)**: 20초마다 자동으로 상승
  - **0~20초**: 50 BPM (Beginner)
  - **20~40초**: 60 BPM (Amateur)
  - **40~60초**: 70 BPM (Pro/Master)
- **박자 구조**: 4/4박자 기반
- **마디 패턴**: 한 마디 연주(노트 발생), 한 마디 휴식(노트 없음) 패턴 반복
- **승리 조건**: 60초를 버티며 최대한 높은 점수 획득
- 조작: PC(키보드 아무 키/마우스), 모바일(터치)

## 노트 시스템
- 노트: `public/ginani.png` 이미지 원형 표현
- **BPM 연동**: 시간이 지남에 따라 낙하 속도와 생성 주기가 BPM에 맞춰 빨라짐
- 휴식 마디에는 노트가 생성되지 않음

## 점수 시스템
- 완벽 (Perfect): **+10 Ginani**
- 좋음 (Good): **+5 Ginani**
- 놓침 (Miss): **-3 Ginani**
- 콤보 시스템: 판정 성공 시 콤보 증가 (점수 가중치는 없음, 기록용)

## UI/UX 설계 기준
- **타이머**: 상단에 남은 시간(초) 표시 (60 -> 0)
- **BPM 인디케이터**: 현재 적용 중인 BPM 표시
- **진행 바**: 전체 60초 중 현재 진행률 시각화
- **결과 화면**: 60초 종료 후 최종 점수 및 판정 통계 표시
- 연주/휴식 마디 시각적 피드백 유지
